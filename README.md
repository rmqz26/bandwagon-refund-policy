## 哪些条件必须同时满足

从搬瓦工官方服务条款里直接提取，退款必须满足以下所有条件：

**第一条（硬门槛）**：这笔订单是新购，不是续费。续费的钱不退。

**第二条（硬门槛）**：账户注册时间在30天以内。老账户直接出局。

**第三条（硬门槛）**：IP地址没有被列入任何黑名单，也没有因为IP被封而向搬瓦工申请过免费换IP。这条卡了很多人——如果你的VPS IP被墙了，先去换IP，换成功后再申请退款。

**第四条（硬门槛）**：从未发起过PayPal争议或信用卡退单。只要发起过一次，永久无资格退款。

**第五条**：当月流量使用量不超过月配额的10%。如果你买了1TB流量的套餐，用了超过100GB就会卡在这里。

**第六条**：退款申请本身不属于滥用性质——也就是说，你不能靠反复注册新账号、反复申请退款来薅羊毛。

说实话，前四条是真正的硬门槛，第五六条对于"用了几天感觉不满意"的普通用户来说基本不会触发。

## 哪些情况肯定退不了

来几个具体场景，对号入座：

**续费订单**：不退。明确写在条款里，续费的钱没有退款资格。

**账户注册超过30天**：不退。哪怕你VPS是刚买的，账户老了就没戏。

**IP被封过且申请了换IP**：大概率不退。官方条款明确，只要你的IP被换过（不管免费还是付费），退款资格直接取消。但有个变通方法：如果IP被封但你还没申请过换IP，先自行付费换一个干净IP，换成功之后再去申请退款——这样IP条件就清了，有机会成功。

**发起过PayPal争议**：永久无资格，而且还会触发账号封禁。切记不要用PayPal争议来代替退款流程，那会让整件事变得更麻烦。

## 退款之后会发生什么

这点很多人没意识到，放在这里重点说。

退款一旦批准，下面三件事同时发生：

- 账户下**所有VPS全部删除**，不是单台，是全部
- 账户下所有数据、快照、备份**永久清除，不可恢复**
- 退款到账

这不是退一台VPS，是清空整个账户。所以在申请退款之前，**一定先把需要保留的数据备份好**。

还有一点：退款之后，这个账户还可以继续用来购买新的搬瓦工服务，但是**不再支持任何形式的退款**。也就是说，退款只能用一次。

## 常见疑问

**Q：年付套餐可以申请退款吗？**

可以，只要满足上面说的条件，年付的钱同样全额退。搬瓦工按账户全额退款，不按使用时长折算。

**Q：退款条件里说消费总额不超过60美元，我买了CN2 GIA-E年付套餐超过这个数，还能退吗？**

实际操作下来，第一次购买即使金额超过60美元，通常也会给退。这条的实际执行没有字面那么严格，主要针对的是反复购买、反复退款的情况。

**Q：可以只退某一台VPS，保留其他的吗？**

不可以。退款是账户级别的操作，要么全退，要么不退，不支持单独退某台机器。

**Q：续费的订单能退吗？**

明确不行。退款政策只适用于新订单。

**Q：申请退款后多久到账？**

一般几十分钟到几小时。支付宝和PayPal通常很快，信用卡可能需要3到14个工作日，取决于你的发卡行。
